# CNN Insects Classificator
**Authors:** Riccardo Serraino, Giuliano Livi

## Setup (one-time)

```
cd path/to/your_folder
```

**Linux**
```bash
python3 -m venv venv
chmod +x run.sh
```

**Windows**
```bat
python3 -m venv venv
```

## Run

**Linux**
```bash
./run.sh
```

**Windows**
```bat
.\run.bat
```
